# План реализации базового шаблона

## 1. Подготовка проекта

### 1.1. Инициализация проекта
- Создать новый проект с помощью Vite: `npm create vite@latest my-app -- --template react-ts`
- Установить зависимости: `npm install`
- Установить дополнительные зависимости:
  - `npm install react-router-dom lucide-react react-icons`
  - `npm install -D tailwindcss postcss autoprefixer`
  - `npx tailwindcss init -p`

### 1.2. Настройка Tailwind CSS
- Создать `tailwind.config.js` с конфигурацией цветов и градиентов
- Обновить `postcss.config.js`
- Добавить директивы Tailwind в `src/index.css`

### 1.3. Структура папок
- Создать структуру папок:
  - `src/components/layout/`
  - `src/components/sections/`
  - `src/components/ui/`
  - `src/pages/`
  - `src/data/`
  - `src/types/`
  - `src/assets/`

## 2. Создание базовых компонентов

### 2.1. Компоненты макета
