# Базовый шаблон для React проектов - Техническое задание

## 1. Обзор проекта

Этот шаблон представляет собой современное React-приложение с TypeScript, использующее Vite в качестве сборщика. Шаблон включает в себя готовую структуру компонентов, систему стилей на основе Tailwind CSS и маршрутизацию.

## 2. Стек технологий

### Основные технологии:
- **React 19** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - строго типизированное расширение JavaScript
- **Vite** - быстрый инструмент сборки и разработки
- **React Router v7** - маршрутизация для одностраничных приложений
- **Tailwind CSS v3** - утилитарный CSS-фреймворк

### UI библиотеки и иконки:
- **Lucide React** - иконки в стиле Lucide
- **React Icons** - дополнительные наборы иконок

### Инструменты разработки:
- **ESLint** - линтер для JavaScript/TypeScript
- **PostCSS** - постпроцессор CSS
- **Autoprefixer** - автоматическое добавление префиксов CSS

## 3. Структура проекта

```
src/
├── assets/                 # Статические ресурсы
├── components/            # Переиспользуемые компоненты
│   ├── layout/            # Компоненты макета (Header, Footer, Layout)
│   ├── sections/          # Секции страниц
│   ├── ui/                # Базовые UI компоненты
│   └── ...                # Другие компоненты
├── data/                  # Данные приложения
├── pages/                 # Страницы приложения
├── types/                 # TypeScript типы
├── App.tsx                # Корневой компонент приложения
└── main.tsx               # Точка входа приложения
```

## 4. Система стилей и дизайн

### Цветовая палитра:
- **brand-blue**: #33a0d8
- **brand-green**: #1d9e3e
- **brand-lightgreen**: #abcb4e
- **brand-orange**: #edb314

### Градиенты:
- **gradient-brand**: линейный градиент от brand-blue к brand-green
- **gradient-fresh**: линейный градиент от brand-lightgreen через brand-blue к brand-green
- **gradient-sun**: линейный градиент от brand-orange к brand-lightgreen

### Анимации:
- **float**: плавающая анимация
- **pulse-slow**: медленный пульс

## 5. Компоненты

### Базовые UI компоненты:
- **Button** - кнопка с вариантами стилей (primary, secondary, outline) и размерами
- **Card** - карточка с возможностью кастомизации
- **Badge** - бейдж для отображения меток
- **PageHeader** - заголовок страницы

### Компоненты макета:
- **Layout** - основной макет с хедером и футером
- **Header** - навигационная панель с адаптивным меню
- **Footer** - нижний колонтитул

### Секции:
- **Hero** - главная секция с призывом к действию
- **Programs** - секция с программами/услугами
- **Benefits** - секция с преимуществами
- **CTASection** - секция с призывом к действию

## 6. Маршрутизация

Приложение использует React Router для навигации между страницами. Основные маршруты:
- `/` - главная страница
- `/programs` - страница с программами
- `/schedule` - расписание
- `/prices` - цены
- `/events` - мероприятия
- `/teachers` - преподаватели
- `/about` - о клубе
- `/contacts` - контакты
- `/try-free` - запись на пробное занятие

## 10. Инструкции по развертыванию

1. Установить зависимости: `npm install`
2. Запустить в режиме разработки: `npm run dev`
3. Собрать проект: `npm run build`
4. Предварительный просмотр: `npm run preview`

## 11. Рекомендации по использованию

1. Использовать компонент Layout для всех страниц
2. Следовать структуре компонентов (ui, sections, layout)
3. Использовать предоставленную цветовую палитру
4. Применять адаптивные классы Tailwind
5. Добавлять новые страницы в App.tsx с соответствующими маршрутами
